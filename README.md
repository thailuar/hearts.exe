# Heart.exe ❤️

Heart.exe é um jogo 2D desenvolvido em JavaScript onde o jogador embarca em uma jornada para resgatar sua namorada, enfrentando inimigos, desafios e obstáculos ao longo de diversas fases.

# Funcionalidades

  - Sistema de movimentação do jogador
  - Combate contra inimigos
  - Sistema de vida e dano
  - Múltiplas fases
  - Objetivos de resgate
  - Interface de usuário (HUD)
  - Controle por teclado
  - Arquitetura modular em JavaScript
  - Tecnologias
  - JavaScript (ES6+)
  - HTML5 Canvas
  - CSS3

# Objetivo do Projeto
Este projeto foi desenvolvido para aprofundar conhecimentos em:
  - Desenvolvimento de jogos 2D
  - Programação orientada a objetos
  - Estruturação de projetos JavaScript
  - Gerenciamento de estados
  - Física e colisões
  - Boas práticas de programação
